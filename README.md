# PlatziVentas

Aplicación de linea de comandos para manejar clientes *CRUD*

## Procesos
- Crear clientes
- Editar Clientes
- Listar Clientes
- Eliminar Clientes

Aplicación realizada en el curso de python 2018 de platzi.

## Instalación

    $ git clone https://github.com/sherzo/pv.git

    $ cd pv
    
    $ pip install .
